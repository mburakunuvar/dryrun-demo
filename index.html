GitHub Copilot Across Surfaces

GitHub Copilot is more than just an IDE extension. It's a comprehensive AI-powered development platform that seamlessly integrates across multiple surfaces to enhance your entire development workflow.

The Four Copilot Surfaces

  • GitHub.com - AI-powered code suggestions, pull request summaries, and intelligent repository insights directly in your browser
  • IDE - Real-time code completions, explanations, and refactoring assistance integrated into your favorite editor
  • CLI - Natural language command suggestions and explanations right in your terminal
  • SDK - Programmatic access to Copilot capabilities for building custom integrations and workflows
